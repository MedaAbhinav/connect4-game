🎮 Connect 4 Game

A web-based Connect 4 game built using HTML, CSS, and JavaScript.
This project demonstrates core JavaScript logic, DOM manipulation, and game development fundamentals.

---

🚀 Features

- 🟡 Two-player gameplay (Red vs Yellow)
- 🎯 Interactive 7x6 grid board
- ⬇️ Piece drop logic (gravity effect)
- 🧠 Win detection:
  - Horizontal
  - Vertical
  - Diagonal
- 🛑 Game stops when a player wins
- 💡 Simple and clean UI

---

🛠️ Technologies Used

- HTML5
- CSS3 (Grid Layout)
- JavaScript (Game Logic)
  
---

🎯 How to Play

1. Click on any column to drop your piece
2. Players take turns (Red and Yellow)
3. First player to connect 4 discs in a row wins
4. Game stops automatically after a win
